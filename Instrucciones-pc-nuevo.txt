Pasos para Ejecutar el Proyecto en el Nuevo Equipo
---------------------------------------------------
1. Ubica la Carpeta autoinventario


2. Coloca la carpeta autoinventario en la ubicación que prefieras en el nuevo equipo (por ejemplo, en el Escritorio).

3. Ve a esa carpeta, entra en la carpeta "\python-3.13.0-embed-amd64" y ejecuta: python.exe get-pip.py

4. Instala las dependencias de requirements: 
cd Scripts\pip.exe
install -r ..\..\requirements.txt


5. Ejecuta el Archivo .bat:

run_autoinventario.bat "C:\Users\albag\Desktop\pynvenv\demo_img" "inventario_salida.docx" "Catarroja" "Holden Caulfield" "dniXXXXXXX" "refXXXXXXX"

Si has creado el archivo run_autoinventario.bat en la carpeta autoinventario, simplemente haz doble clic en run_autoinventario.bat.
Este archivo por lotes debería ejecutar el archivo main.py utilizando el Python embebido en la carpeta python-3.13.0-embed-amd64.
Si no has creado este archivo .bat, sigue los pasos a continuación.
